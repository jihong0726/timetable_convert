功能 4 · 按日期查代码
使用【三区域输入】：
- 日期 + 星期；
- 姓名列表（支持「英文名 + 中文名」格式）；
- 时间 / 代码（如 9:00 / 15:00 / AL / OFF / REST 等）。
再选择某个日期 + 某个代码，查看那天对应代码是谁。
1. 按三块粘贴原始内容 结构需要和飞书表格列顺序一致
① 日期 + 星期
② 姓名列表
③ 时间 / 代码
提示：请先粘贴完整三块内容，再点击「查询」，系统会自动填充可选的日期与代码下拉列表。
2. 查询结果 支持查看某日某代码的所有人员
统计
这里会显示：当前选择的日期、代码，以及匹配到的总人数。
人员名单
这里会显示：所有在该日期使用该代码 / 班次的姓名列表。
